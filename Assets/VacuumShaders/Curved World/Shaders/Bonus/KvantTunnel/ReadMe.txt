Folder contains two KvantTunnel shaders (Line, Surface) modified for Curved World.

How to use
1. Download KvantTunnel project from github: https://github.com/keijiro/KvantTunnel
2. Replace shaders in Assets/Kvant/Tunnel/Shaders



KvantTunnel
KvantTunnel is a GPU accelerated fractal tunnel renderer for Unity.

System Requirements
Unity 5.1 or later versions.

Compatibility
KvantTunnel needs a 4-component floating-point texture format. Basically, it means it runs only on desktop/laptop-class GPUs.
